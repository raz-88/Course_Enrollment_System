Manage Topics for Course

Select a course, then add / edit topics. Each topic has a max capacity (how many people can enroll) and you can see how many members are already assigned in groups.

For group topics this is number of groups allowed; for individual topics it's total seats.
If Group is selected, users on the portal will only see the Group enrollment option for this topic.
Group min / max define how many members can join a group. Admins set these values.
